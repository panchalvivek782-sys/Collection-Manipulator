# ## Welcome to the Student Data Organizer!
#
# This repository contains a complete Python command-line utility.
# The tool lets you manage and track student details efficiently.
#
# ## Features
#
# * **Add Student**: Save names, ages, grades, birthdays, and subjects.
# * **Display All**: View all stored student profiles at once.
# * **Update Records**: Edit details easily while keeping existing data.
# * **Delete Option**: Remove old student data from the system instantly.
# * **Subject Tracker**: List all unique subjects taken by active students.
# * **Data Check**: Stops you from using the same ID twice.
#
# ## How It Works
#
# 1. Run the Python file in your favorite terminal window.
# 2. Pick an option from 1 to 6 on the menu screen.
# 3. Type in the requested information when prompted by the app.
# 4. Watch your student records update immediately in real time.
#
# ## Requirements
#
# * Python 3.x
